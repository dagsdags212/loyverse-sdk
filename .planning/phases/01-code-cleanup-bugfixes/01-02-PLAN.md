    Target content:
    ```python
    class MerchantEndpoint(BaseEndpoint, RetrieveMixin):
        path = "merchant"

        async def retrieve(self):
            data = await self._get(self.path)
            return Merchant.model_validate(data)
    ```

    Steps:
    1. Remove `id: str` from the `retrieve` method signature.
    2. Replace `return await super().retrieve(id, model=Merchant)` with:
       ```python
       data = await self._get(self.path)
       return Merchant.model_validate(data)
       ```
    3. No import changes needed — `Merchant` is already imported (line 5), `BaseEndpoint._get` is inherited, and no `from` import is needed for `_get` since it's a method on `BaseEndpoint` which `MerchantEndpoint` inherits.
    4. The `RetrieveMixin` import can be kept — it's harmless and removing it from line 2-4 would change more than needed. Alternatively, since `retrieve()` no longer calls `super().retrieve()`, `RetrieveMixin` is technically unused but keeping it is cleaner than removing it (the class still inherits from it for interface consistency).
